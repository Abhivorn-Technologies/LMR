# LMB Insurance Brokers

Premium corporate website for LMB Insurance Brokers — composite insurance broking firm.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

Next.js 15 · TypeScript · Tailwind CSS v4 · Framer Motion · GSAP · Lenis · React Hook Form · Zod · Embla Carousel

## Design

Black + Cyan premium dark theme. Temporary logo at `public/assets/logo/lmb-logo-temporary.png`.

## Content

Business content placeholders marked `(Content Required from Client)` until brochure details are provided.
